5 HTML tags I've learned

Paragraphs
Using the tag <p></p> will create a text paragraph
Quotes
Using the tag <q></q> will create a small quote embedded within other content
Block quotes
Using the tag <blockquote></blockquote> will create a extended quotation that would sit by itself outside of other paragraphs
Highlight text
Using the tag <mark></mark> will highlight a piece of text
Emphasis
Using the tag <em></em> will add emphasis to important text
