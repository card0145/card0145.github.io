Bootstrap 101

Bootstrap is a free open-source css framework developed by Mark Otto and Jacob Thornton. It was first created as a solution for issues such as inconsistencies and high maintenance found when using various libraries for interface development.

Below, we will go through some of Bootstraps features.

Grids

One of bootstrap's most powerful features is it's responsive twelve column grid system which features six default breakpoints.

Grid Breakpoints

Size Class Pixels
xsmall
small sm ≤ 576px
medium md ≤ 768px
large lg ≤ 992px
xlarge xl ≤ 1200px
xxlarge xxl ≤ 1400px

These breakpoints easily allow us to control widths of particular elements according to certain screen sizes. As shown below, the widths of both paragraphs change according to specific breakpoints.
